hexo 博客简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页

安装教程

安装 Node.js

在官网下载Node.js.
如果有多个 Node 版本，使用nvm管理 – 一个 nodejs 版本管理工具

nvm 安装

如果已有 node，卸载后安装 nvm
打开 CMD，输入命令 nvm ，安装成功则列出各种命令。

下载 node.js 时候可能限速，可以修改 nvm 安装路径下的”settings.txt“文件提速。
nvm 安装路径，找到 settings.txt 文件

终端输入 nvm list available，查看可以安装的版本。
选择一个版本安装， nvm install "版本号"。
输入nvm list 查看已经安装的 node，*号表示当前使用的 node 版本。
更换版本，nvm use "版本号"。

node.js 安装

如果不使用 nvm,直接在官网下载安装 node 即可。

安装完成后打开 cmd，输入node -v 和 npm -v查询 node 和 npm 的版本号。

Hexo 命令速查表

命令 说明
hexo init [folder] 初始化一个项目
hexo new “文章标题”(hexo n) 创建一篇新文章
hexo new draft “文章标题” 创建一篇草稿文章
hexo new page “页面名称” 创建一个新页面
hexo generate(hexo g) 生成静态文件
hexo server(hexo s) 启动本地服务器预览网站
hexo deploy(hexo d) 部署网站到远程仓库
hexo clean 清除缓存文件
